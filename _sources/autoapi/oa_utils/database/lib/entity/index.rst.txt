.. py:module:: oa_utils.database.lib.entity

.. autoapi-nested-parse::

   This module defines a database schema.

   In order to store and retrieve data from the database in a more
   user-friendly manner, we have to associate every database document
   type with a class defining the structure of that document.

   Check mongoengine documentation for more information on that subject.
