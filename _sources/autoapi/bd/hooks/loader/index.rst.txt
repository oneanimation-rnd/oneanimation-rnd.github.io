.. py:function:: get_searchpath(root_dir)

   Return a list of all subdirectories under the root directory.

   This function goes to every subdirectory recursively and appends
   it path to the resulting list.

   :param root_dir: A parent directory.
   :type root_dir: str

   :returns: A list of subdirectories including the root directory.
   :rtype: list[str]
