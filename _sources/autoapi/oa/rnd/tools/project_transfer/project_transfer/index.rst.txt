.. py:module:: oa.rnd.tools.project_transfer.project_transfer

.. autoapi-nested-parse::

   Transfers assets from another project to the current project via checking it in.
   Reference paths are re-mapped to point at the new project.

   There a few gotchas right now.
       - It only transfers the latest version.
       - It assumes the asset doesn't exist, so there's no handling for that.
       - It doesn't transfer over textures.
       - It re-points reference paths to v001 since those will be new transfers anyways.
