.. py:class:: DailyReminder(hour, func, interval=30000, parent=None)

   A reminder that schedules a time and periodically checks if it goes past it.
   Once it does it triggers a function and re-schedules a new time.

   .. py:method:: register(hour, func)
      :staticmethod:

      Adds a new daily reminder everytime it passes the supplied hour.

      :param hour: A value from 0-23.
      :type hour: int
      :param func: The function it will run.
      :type func: function


   .. py:method:: __del__()


   .. py:method:: check_schedule()



.. py:function:: get_next_timing(hour)

   Calculates the time for the supplied coming hour.

   :param hour: A value from 0-23.
   :type hour: int

   :returns: A QDateTime object for the next timing.
