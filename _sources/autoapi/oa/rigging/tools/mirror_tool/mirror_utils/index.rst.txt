.. py:function:: mirror_objs(objs, axis, flip_x=False, flip_y=False, flip_z=False, duplicate=False)

   Mirrors supplied objects without effecting their scale.
   Multiplies transforms with the same axises used by Maya's mirror joints tool.

   :param objs: A list of exiting object names.
   :param axis: A key from the variable mirror_axis.
   :param flip_x: Flips the object's x position.
   :param flip_y: Flips the object's y position.
   :param flip_z: Flips the object's z position.
   :param duplicate: Duplicates the object then mirrors it, otherwise does it with the supplied object.
