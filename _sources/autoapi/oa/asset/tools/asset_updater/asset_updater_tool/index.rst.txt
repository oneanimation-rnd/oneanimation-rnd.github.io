.. py:module:: oa.asset.tools.asset_updater.asset_updater_tool

.. autoapi-nested-parse::

   A simple tool to pop-up a dialog when old asset versions are detected in the scene.
   Nested assets are ignored.
   A callback is included to trigger this after a file finishes opening.

   Author:
       Jason Labbe
