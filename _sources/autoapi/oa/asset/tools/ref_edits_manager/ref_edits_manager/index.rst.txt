.. py:module:: oa.asset.tools.ref_edits_manager.ref_edits_manager

.. autoapi-nested-parse::

   Reads a reference's edits so that the user can choose which to remove.
   This reads and applies edits significantly faster than Maya's native tool.
   Sometimes Maya's native tool won't remove specified edits, whereas this one does.

   Author:
       Jason Labbe

   .. todo:: - Support removing edits on multiple references in a single action.

   Limitations:
       - Failed edits are not included and will be removed if edits are re-applied.
       - Ignores nested references.

   Usage:
       import ref_edits_manager
       ref_edits_manager.launch()
