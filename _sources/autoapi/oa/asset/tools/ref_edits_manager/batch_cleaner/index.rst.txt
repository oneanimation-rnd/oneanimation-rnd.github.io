   Author:
       Jason Labbe

   Usage:
       import batch_cleaner
       batch_cleaner.clean_all_ref_edits(["*.colorSet*"])  # Cleans current open scene.
       batch_cleaner.batch_run(["/path/scn.ma"], "/outputDir", ["*.colorSet*"], True)  # Opens and cleans multiple scenes and prints benchmarks info.
