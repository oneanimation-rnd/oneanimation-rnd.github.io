.. autoapi-nested-parse::

   Automates how much spin a wheel rotates while traveling through space.
   First the asset must be setup with a master, directional, and wheels objects.
   The sim will run in memory then apply it onto the objects by setting keyframes.

   Author:
       Jason Labbe
