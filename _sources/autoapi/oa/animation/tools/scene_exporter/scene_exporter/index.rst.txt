.. py:function:: export_scene(project, ep, seq, shot_task_type, shot_list, notes, quiet=False, reload_scene=False)

   Exports scene onto the server.

   :param project: Project name.
   :type project: string
   :param ep: Episode name.
   :type ep: string
   :param seq: Sequence name.
   :type seq: string
   :param shot_task_type: ANM, LA.
   :type shot_task_type: string
   :param shot_list: A list of shot names with their zero padding.
   :type shot_list: string[]
   :param notes: Comments to include in log.
   :type notes: string
   :param quiet: Pops up messages if True.
   :type quiet: bool
   :param reload_scene: Reloads scene after export is done. This will be overridden when quiet is False.
   :type reload_scene: bool

   :returns: Full path to exported scene file on the server.
