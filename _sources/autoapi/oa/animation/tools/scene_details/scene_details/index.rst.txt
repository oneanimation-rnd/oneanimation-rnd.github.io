.. py:module:: oa.animation.tools.scene_details.scene_details

.. autoapi-nested-parse::

   An upgraded version of a previous tool that list all of our assets in the scene.
   It also gives the ability to convert any asset to GPU to increase the scene's performance.
   An object set and layer viewers are bundled in it to prevent using the Outliner.

   Author:
       Jason Labbe
