   Constraint tool

   Interface to unify and manage constraints in animation scenes.

   Author:
       Jason Labbe

   Usage:
       tool = ConstraintTool()
       tool.run()

   Constraint's structure:
       - obj (Object that is being constrain)

       - constraint_nul (obj is parent constraint to this)
           - offset_loc (Used to set keys on offsets or weights to targets)

       - target
           - target_loc (Constraining constraint_nul to this. Shares same transform as obj, so that there's less flipping.)
