.. py:function:: cut_mesh_by_curves(mesh_objs, curve_objs, close_mesh=False, triangulate=False)

   Splits meshes with curves. Cutting is based on the current view.

   :param mesh_objs: Collection of mesh objects to split
   :type mesh_objs: string[]
   :param curve_objs: Collection of curve objects that will cut
   :type curve_objs: string[]
   :param close_mesh: Closes the new mesh objects so there won't be any holes
   :type close_mesh: bool
   :param triangulate: Triangulates the new mesh objects
   :type triangulate: bool

   :returns: A list of all new mesh objects
