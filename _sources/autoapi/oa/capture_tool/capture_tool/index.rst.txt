.. py:module:: oa.capture_tool.capture_tool

.. autoapi-nested-parse::

   Uses Qt's framework to capture the screen then open an editor to paint on the images.
   Later the images can be exported and be stitched together using ffmpeg.

   Author:
       Jason Labbe
