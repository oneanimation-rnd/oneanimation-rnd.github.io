##########################
Frequently asked Questions
##########################


Q. How do I create a new github repo
    New Github repos for the organization are created by the Head of RnD. You
    are always welcome to create repositories under your own username, please
    keep forks of studio repositories private.


Q. I cannot push to github with my username and password. Please help.
    Github does not support the use of passwords for repository access any
    more. Please set up an ssh key to generate an access token.
