oa.rigging.libs.control_objects.control_utils.export_preset(curve, output_dir, preset_name=None)

Exports a shape to json file.

Parameters

  • curve – An object with curve shapes.

  • output_dir – Full path to a directory to export preset to.

  • preset_name – The name to set the preset’s file. If nothing is supplied, it uses the object’s name.

oa.rigging.libs.control_objects.control_utils.import_preset(preset_name, size=None, name='', parent='', align='', color_index=None, input_name=False, add_nul=True)

Imports an exported shape via json format.

Parameters

  • preset_name – Preset name to import.

  • size – Size multiplier. If nothing is supplied it tries to use the value from its settings file.

  • name – If an empty string is supplied it will use the preset’s name.

  • parent – The object to parent the new control to.

  • align – Aligns the new control this object.

  • color_index – Passing as an integer (between 1-31) will use Maya’s internal colorIndex. Passing a 3 float list (between 0-1) will use rgb.

  • input_name – Prompts user to enter a name.

  • add_nul – Adds an empty group on top of the ctrl to zero it out.

Returns

A list of the control’s nul and object.
