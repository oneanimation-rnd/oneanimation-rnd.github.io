oa.rigging.tools.auto_rigging.rigging_utils.get_axis_dir(obj, axis, direction)

Determines in world space what direction an object’s axis is facing. This is used mostly to determine if an axis was mirrored or not.

Parameters

  • obj (string) – An object’s name.

  • axis (string) – The object’s local axis to check with. Value can be x, y, or z.

  • direction (string) – The world space direction to see what way the axis is facing. Value can be x (left & right), y (up & down), or z (backwards & forwards).

Returns

Either 1 or -1 depending on the direction the axis is facing. x: 1=left, -1=down y: 1=up, -1=down z: 1=forwards, -1=backwards

Automates setup to switch an object to a different rotational space.

Parameters

  • obj (string) – The object to switch space.

  • attr_holder (string) – An object to hold the space switch attribute.

  • prefix (string) – The prefix to use for all object names.

  • spaces (string[]) – A list of space names that displays in the attribute.

  • parents (string[]) – A list of objects the space parents to.

  • world_space_index (int) – The top space to parent to.

  • local_space_index (int) – The space that all the others will be point constraint to, or None to do nothing.

  • attr_name (string) – The attribute’s internal name.
