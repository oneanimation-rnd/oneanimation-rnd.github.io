 oa.rigging.tools.edit_skin.edit_skin

Author: Jason Labbe

Run editSkin() on a mesh with a skinCluster to enable edit mode. The skinCluster and constraints will be disabled so you can tweak joint positions. Press ctrl+alt+q to exit edit mode and restore everything.
