 oa.general.libs.code_bouncer

A module that uses basic validation to block users from running code we don’t want them access to.

Author:

Jason Labbe

Example of usage:

from oa.general.libs import code_bouncer

# Raises exception if validation fails. with code_bouncer.CodeBouncer(trusted_groups=[“rnd”], password=”bW9yb3g=

“):

# Validation passed and continues execution. print “Hello world!”
