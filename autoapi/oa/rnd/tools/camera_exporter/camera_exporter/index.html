oa.rnd.tools.camera_exporter.camera_exporter._create_camera_instance(src_cam_shape)

Create camera instance in world space.

Create a transform on the scene root level and put the provided camera shape underneath it. Parent constraint that transform to the original camera to inherit all the transformations in world space.

Returns

path to the new camera.
