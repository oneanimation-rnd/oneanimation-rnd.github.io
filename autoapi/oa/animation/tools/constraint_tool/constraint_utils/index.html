oa.animation.tools.constraint_tool.constraint_utils.create_offset(obj, target, constraint_type, style, size, color, group=global_set)

Creates an offset control and nul. offset_ctrl=Object the animators will use. The object constraints to this. offset_nul=Constraints with the target.

Parameters

  • obj – The object to constraint.

  • target – The object to drive the constraint.

  • constraint_type – enums.ConstraintType.

  • style – The offset’s shape style.

  • size – The size of the offset ctrl.

  • color – An rgb 3-float list to color the offset ctrl.

  • group – The object set the offset ctrl should be added to.

Returns

A list of the offset nul and offset ctrl.
