 oa.animation.tools.key_breakdown.key_breakdown_tool

This tool allows you to quickly set keys to create breakdowns that blend between the previous and next keys. The slider can be used to get more precise control, or presets are available.

There are different layer of filters to work with.

1 - It will key all keyable attributes that have a key. 2 - If there’s any selected attributes on the channel box it will only key on them. 3 - If there’s any selected keys from the graph editor it will only key on them. This overrides the channel box filter. 4 - There’s an option on the interface to key only on translation or rotation. If either of these are on it will override both the channel box and graph editor filters.

Jason Labbe
