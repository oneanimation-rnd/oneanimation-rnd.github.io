oa.animation.tools.key_breakdown.key_breakdown_utils.set_keys_info(keys_info, bias, as_breakdown_key=False)

Sets a key on the current frame with a value that biases from an object’s previous or next keys.

Parameters

  • keys_info (dict) – A data structure to parse through.

  • bias (float) – A value from -100 to 100, to determine if the value should prefer the previous or next keys. Supplying a value beyond 100 can be used to ‘overshoot’ the new value.

  • as_breakdown_key (bool) – Makes new keys green.
