oa.animation.tools.mirror_animation.mirror_animation_utils.collect_auto_mapping()

This should run when the asset is in its default pose. It will check all controls to determine if they should be swapped or flipped on itself during mirroring. It also collects which axes need to be flipped for swapping objects.

Returns

A dictionary to represent mapping data.

oa.animation.tools.mirror_animation.mirror_animation_utils.set_auto_mapping()

This should run when the asset is in its default pose. Quick function to collect mapping and set it on the mapping node.

oa.animation.tools.mirror_animation.mirror_animation_utils.mirror_anim(nsp, selection_only, time=None, skip=[])

The core that mirrors the animation. Instead of using matrix math, it uses anim curve functions to manipulate them.

Parameters

  • nsp (string) – The namespace to operate on.

  • selection_only (bool) – Filters only selected controls when True.

  • time (float[]) – A two item tuple to operate on keys that lie in this time range.

  • skip (string[]) – A list of control names to ignore.
