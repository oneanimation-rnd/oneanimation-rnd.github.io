setup_viewport_for_capture(render_cam_name, enable_dof, enable_ao, display_textures, viewport_mode, use_lights, show_settings, eval_mode='serial')
Parameters

  • render_cam_name (string) – Name of the camera that’s used for capturing.

  • enable_dof (bool) – Enables DOF on capture.

  • enable_ao (bool) – Enables ambient occlusion on capture.

  • display_textures (bool) – Enables textures to be displayed.

  • viewport_mode (string) – Viewport mode of the camera that’s used for capturing.

  • use_lights (bool) – Uses all lights on True.

  • show_settings (dict) – A dictionary of data to determine what should be hidden in the view.

  • eval_mode (string) – The evaluation type that the capture will run in.
