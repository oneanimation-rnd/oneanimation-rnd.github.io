Reformat the code so it is possible to apply it to the objects without any errors.

For .take file structure reference see https://www.sidefx.com/forum/topic/4691/

Change all the GUID paths to real node paths or remove references if the node doesn’t exist in the scene.

Parameters

  • take_creation_code (str) – .take file contents.

  • take_name (str) – the name of the take to format.

Returns

Reformatted code on success.
