oa.asset.libs.import_utils.switch_resolution(ref_path, res, latest=False, force=False)

Replaces a reference with a different resolution.

Parameters

  • ref_path (string) – A resolved reference path.

  • res (string) – The resolution to switch to.

  • latest (bool) – It will use the latest version on True, otherwise it will try to stick with the same version.

  • force (bool) – Normally it does nothing if the resulting path is the same, but force enabled will load it regardless.

Returns

The new resovled reference path.
