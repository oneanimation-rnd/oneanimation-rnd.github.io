oa.asset.libs.maya_assets.parent_geo_to_asset(resolution_type, objs=None, parent_obj=None)

Parents a geometry under the rig’s NOTOUCH_NUL. If resolution is “RDR”, all supplied objects will be added to the render set.

Parameters

  • objs (string[]) – List of objects to parent. If None, defaults to current selection.

  • resolution_type (string) – A nul with a corresponding name will be created under NOTOUCH_NUL, where the meshes will be stored. Typically “RDR” for hiRes or “ANI” for lowRes.

  • parent_obj (string) – If supplied, objects will parent to an empty nul that has a constraint to this object.
