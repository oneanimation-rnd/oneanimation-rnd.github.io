 oa.asset.tools.asset_exporter.asset_exporter

Deals with exporting and publishing an asset onto the server.

The exporter outputs the following:

  • Original maya scene

  • RDR maya scene

  • Annotation file

  • maINF descriptor file

  • Base pose alembic cache

  • Thumbnail

  • GPU cache (except with CHRs)

  • GPU maya scene (except with CHRs)

Author:

Jason Labbe

Usage:

run_exporter()

oa.asset.tools.asset_exporter.asset_exporter.check_in_file(transfer, local_path, publish_path, annotation='', remove_local_file=True, **metadata)

Creates an empty file to the specified path on the server. Still need to write to this file.

Parameters

  • transfer (transfer_utils.Transfer) –

  • local_path (string) – Full path to a file that will be transfered.

  • publish_path (string) – Full path to file to save to.

  • annotation (string) – Creates a log file. Log file is not created if empty.

  • remove_local_file (bool) – Removes local path after it’s done the check-in.

  • metadata (dict) – Extra data to put in annotation file.

Returns

A path to the new file.
