oa.asset.tools.asset_creator.asset_creator.create_new_directory(asset_type, project, name, category=None, level=None)

Creates a new asset directory and its structure on the server via ftp. Also checks-in its initial entry into the database.

Parameters

  • asset_type (string) – CHR, CAM, PRP, LOC.

  • project (string) – Project’s name.

  • name (string) – Asset’s name.

  • category (string) – Asset’s category (for PRP and LOC assets only).

  • level (string) – Asset’s location level (for LOC assets only).

Retuns:

A full path to the new directory on success.
