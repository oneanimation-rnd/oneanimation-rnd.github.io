 oa.asset.tools.asset_cacher.asset_cacher

Creates alembic caches for assets.

This is only meant to be used on the farm, and not locally.

Connecting a shape has been optimized significantly by George “I only need a day to fix it” Samushyia. Before it was done purely with script and was error-prone and slow, but now it uses nodes to make it work in real-time. Thank you for saving my ass!

Author:

Jason Labbe
