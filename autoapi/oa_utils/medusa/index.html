checkIn(localFilePath, destFilePath, annotation='', maxVersions=-1, forceUnlock=False)

CheckIn localFile to the server and create new version in ‘OLD’ folder.

Before uploading file to the server it checks if the file with the same name is already on the asset drive. If so, it moves this file to an ‘OLD’ folder and adds version number in it’s name. After that it uploads local file to remote destination.

Parameters

  • localFilePath (str) – path to a file on a local computer.

  • destFilePath (str) – path to a file or folder on the asset drive.

Kwargs:

annotation(str): comments. maxVersions(int): a maximum amount of versions to store in the “OLD” dir ( -1 = infinite )
